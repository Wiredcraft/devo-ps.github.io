devo.ps transactional email templates
=====================================

## Manual generate
- template/

Use [premailer](https://github.com/premailer/premailer) to convert css into html inline-css.

```
$ cd template
$ premailer action.html > out-action.html
```

## Auto generate
- grunt-template/

```
$ cd grunt-template
$ npm install
$ grunt
```
